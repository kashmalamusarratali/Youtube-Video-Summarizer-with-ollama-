# Youtube-Video-Summarizer-with-ollama-
A simple web-based application that summarizes YouTube videos using their transcripts and a Large Language Model (LLM).   Built with Python, Gradio, YouTube Transcript API, and Ollama (LLaMA).


##  Features

- Accepts any YouTube video URL
- Automatically fetches video transcripts
- Generates a concise and clear summary
- Easy-to-use Gradio web interface
- Runs locally using Ollama (no paid API required)



## Tech Stack

- Python
- Gradio
- YouTube Transcript API
- Ollama (LLaMA 3.2)
- OpenAI-compatible API interface
